# React Todo App

A simple React-based todo list application for managing tasks. Add, edit, and delete todos with a clean UI.

- Clone repo
- Run `npm install`
- Start with `npm start`

to create a fresh React app:

npx create-react-app my-todo-app
cd my-todo-app
npm start
